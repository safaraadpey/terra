# Terra

**Terra** is the foundational semantic substrate that transforms the human-centric web
into a structured, meaningful, and directly executable environment for intelligent agents.

The web was built for humans: pages, visuals, interfaces, and interactions.
Terra turns the web into ground that intelligent systems can understand, traverse, and act upon.

---

## What Terra Is

Terra is not a browser.
It is not a crawler.
It is not a chatbot.

Terra is a **semantic ground layer** that sits beneath the web,
making existing websites readable and executable for AI systems
without requiring changes to those websites.

---

## Core Capabilities

Terra enables three fundamental capabilities:

### 1. Read  
Access and normalize web content beyond visual presentation.

### 2. Understand  
Translate web pages into structured semantic meaning using **WebML**  
(entities, relations, and actions).

### 3. Act  
Enable intelligent agents to perform real, controlled actions on the web
through an execution runtime.
